科目三考试技巧

作者: YuZhangWang 日期: 2020-05-14 阅读时间:2.4 分钟 字数:639 分类: 生活 阅读量: -

注意事项

车头上车，待机入道，开门适中，人入门关。三事毋忘，坐椅视镜，系安全带，礼貌问候，仪容仪表。夜间灯光，检查复位，模拟开行，听令毋抢。考试结束，得单挂科，得卡通过，二次开门，相机而出，出既关门，逆向而出。毋坐黑车，防止受骗

灯光

下面将进行模拟夜间行驶场景灯光的考试，请在5秒内作出相应的灯光操作:
1.打开前照灯

2.交替使用远近光灯(不允许出现除交替使用远近光灯外的其他灯光变化)
(1-1)夜间通过急弯
(1-2)夜间通过坡路
(1-3)夜间通过拱桥
(1-4)夜间通过人行横道
(1-5)夜间通过没有交通信号灯控制的路口(1-6)夜间超车

3.开近光灯
(1-1)夜间近距离跟车行驶
(1-2)夜间会车
(1-3)夜间直行通过路口
(1-4)夜间左右转弯
(1-5)夜间在有灯光的道路上行驶
(1-6)夜间在照明良好的道路上行驶

4．开远光灯
(1-1)夜间在没有路灯的道路上行驶
(1-2)夜间在照明不良的道路上行驶

5.开示廓灯和危险报警闪光灯
(1-1)路边临时停车
(1-2)车辆发生故障和交通事故

6.关闭所有灯光

驾考口诀

起步

1.打左转向灯
2.踩离合
3.踩脚刹
4.挂1挡
5.松手刹
6.侧身左右观察
7.抬离合，找联动点
8.松脚刹
9.左后视镜观察
10.调方向
11.轻加油门
12.关方向灯

变道

1.打灯(左/右)
2.等3秒
3.左/右后视镜观察
4.调方向
5.关方向灯

靠边停车

1.打右转向灯
2.踩离合
3.轻踩脚刹
4.看右后视镜
5.调方向
6.拉手刹
7.挂空档
8.抬离合
9.松脚刹
10.关方向灯

右转

1.打右转向灯
2.踩离合
3.踩脚刹(20以下)
4.松脚刹
5.挂二档
6.抬离合
7.看右后视镜
8.左右观察
9.调方向

掉头

1.打左转向灯
2.踩离合
3.踩脚刹(20以下)
4.松刹车
5.挂二档
6.抬离合
7.看后视镜
8.左右观察
9.调方向

通过

1.踩离合
2.踩脚刹(20/30)
3.松脚刹
4.挂匹配档位
5.抬离合
6.轻加油门
7.左右观察
