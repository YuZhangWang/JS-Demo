【我的EDC】第五期——电脑

黑科技 生活  •   阅读量  0  •  阅读时间: 0.8分
💤YuZhangWang  2020-05-14

不玩游戏情况下，一颗i5，6代，甚至是低压的U，已经可以满足我敲代码的需求了。固态+8G内存，日常运行不卡不慢，足够了。
管理我电脑的是火绒和Men Reduct，一个防病毒一个调内存，都在后台帮我默默工作着。

一般不点击图标运行的软件将它藏起来，让他们帮我自动管理，协调电脑。

Google Chrome装了一个常用的插件，哔哩哔哩助手帮我优化B站(默认关闭弹幕、优化播放窗口)，Stylish给我心仪的B站主题，广告终结者和Adblock Plus帮我扫清我上网遇到的一切广告(包括各大网
站视频开头广告)。

一块副屏，平常监视B站粉丝数。使用的时候可以在左边刷动态
